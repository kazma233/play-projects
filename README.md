# 项目列表

- **deploygo**: DeployGo 是一个轻量级的 CI/CD 工具，使用容器（Docker/Podman）进行构建，并通过 SFTP 部署到远程服务器。
- **picstash**: 基于 Git 的图床存储方案，配合瀑布流图片展览。
- **kttools**: 自用，基于Tauri开发的工具箱
- **ftpgo**: 一个基于 Go + Vue3 + Tailwind CSS 的轻量级文件管理器，支持 FTP 风格的文件浏览、上传、下载和管理功能。
- **backupgo**: 定时将你的目录/文件压缩，然后上传到OSS上。支持前置/后置自定义命令，也支持内置 Postgres、MongoDB 和 Docker volume 备份源。
- **agent-session-hub**: 本地桌面工具，用于浏览 Codex、Claude Code、OpenCode 的历史会话，并导入到另一个工具。
