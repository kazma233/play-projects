# 项目列表

- [agent-history-hub](./agent-history-hub): 本地桌面工具，用于浏览 Codex、Claude Code、OpenCode 的历史会话，并导入到另一个工具。
- [backupgo](./backupgo): 定时将你的目录/文件压缩，然后上传到 OSS。支持前置/后置自定义命令，也支持内置 Postgres、MongoDB 和 Docker volume 备份源。
- [deploygo](./deploygo): 一个轻量级的 CI/CD 工具，使用容器（Docker/Podman）进行构建，并通过 SFTP 部署到远程服务器。
- [ftpgo](./ftpgo): 一个基于 Go + Vue 3 + Tailwind CSS 的轻量级文件管理器，支持 FTP 风格的文件浏览、上传、下载和管理功能。
- [kt-img](./kt-img): 基于 Tauri + Vue 3 + Rust 的本地图片压缩与转码工具，支持预览、压缩、缩放和批量导出。
- [kt-port](./kt-port): 基于 `iced` 的端口查看和进程终止工具，支持筛选、排序、详情查看和终止选中进程。
- [kttools](./kttools): 自用，基于 Tauri 开发的工具箱。
- [picstash](./picstash): 基于 Git 的图床存储方案，配合瀑布流图片展览。
